# mock-server
利用electron构建的桌面程序,根据自定义路径和数据格式,启动服务器,响应数据

> An electron-vue project

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build


```
